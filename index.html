Problem Statements

Choose ONE problem statement for your team

1. Data Cleaning & EDA

  • Clean a raw dataset
  • Handle missing values and duplicates
  • Identify trends and insights
  • Create comprehensive EDA report

2. Business Data Analytics

  • Analyze sales or business data
  • Create insights for decision-making
  • Provide data-driven recommendations
  • Build business intelligence reports

3. Dashboard & Visualization

  • Build an interactive dashboard
  • Display KPIs and charts
  • Tell a clear story using visuals

  • Analyze features and results

  • Validate model performance

Rules & Regulations

  1. Participants must register in teams (1-4 members per team).
  2. Each team must select only one problem statement.
  3. All submissions must be original work.
  4. Use of tools such as Python, SQL, Power BI, Excel, Tableau is allowed.
  5. Plagiarism will lead to immediate disqualification.
  6. All participants must submit their project before the deadline.
  7. The decision of the judges will be final and binding.
